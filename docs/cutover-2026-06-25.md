# SDK → AFL Forge Cutover — 2026-06-25

On 2026-06-25, Brokkr (the autonomous forge) was migrated off the Claude Agent SDK and onto the native `@brokk/afl` agent-loop. The switch eliminates the SDK abstraction layer, giving Brokkr direct control over the agent loop and reducing external dependencies. All forge workflows continue to operate as before; only the underlying runtime has changed.
